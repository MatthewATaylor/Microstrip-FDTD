# Microstrip-FDTD

 A custom finite difference time domain (FDTD) solver for Maxwell's equations designed to simulate microstrip radio-frequency circuits.

 ## Features

 * Perfect electrical conductors
 * Lossless and lossy dielectrics
 * Lumped elements
    * Resistors
    * Resistive voltage sources
    * Inductors
* Convolutional CFS-PML absorbing boundaries
