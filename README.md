# Microstrip-FDTD

 A custom finite difference time domain (FDTD) solver for Maxwell's equations designed to simulate microstrip radio-frequency circuits. Developed as part of the final project for MIT class 6.2300 (Electromagnetic Waves and Applications) and used to simulate linearly polarized and circularly polarized patch antenna designs. S11 parameters simulated using the FDTD solver were in close agreement with HFSS simulations and fabricated designs.

 ## Features

 * Perfect electrical conductors
 * Lossless and lossy dielectrics
 * Lumped elements
    * Resistors
    * Resistive voltage sources
    * Inductors
* Convolutional CFS-PML absorbing boundaries
